# DeepSeek vs GLM: 国产大模型对比

## 核心定位
- **DeepSeek**：开源先锋，推理能力强，性价比极高
- **GLM (智谱)**：企业级服务，中文优化，API 稳定性高

## 技术对比

| 维度 | DeepSeek-V3/R1 | GLM-4/4-Plus |
|------|---------------|-------------|
| 开源程度 | 🔥 完全开源（MIT） | 🟡 部分开源 |
| 推理能力 | 🔥 R1 推理链极强 | 🟢 良好 |
| 中文能力 | 🟢 优秀 | 🔥 原生中文优化 |
| API 价格 | 💰 极低（输入 ¥1/M token） | 💰 中等 |
| 上下文长度 | 64K | 128K |
| Embedding | 支持 | 支持（embedding-3） |
| 企业支持 | 社区驱动 | 🔥 官方企业服务 |

## 适用场景

**选 DeepSeek 当：**
- 成本敏感的大规模应用
- 需要强推理能力（数学、逻辑）
- 想自部署模型

**选 GLM 当：**
- 企业级生产环境（需要 SLA）
- 中文原生场景
- 需要稳定的 Embedding 服务
