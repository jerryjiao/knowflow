# Claude vs GPT: AI 编程助手对比

## 核心定位
- **Claude (Anthropic)**：长上下文、代码理解深度、安全优先
- **GPT (OpenAI)**：生态成熟、插件丰富、多模态能力强

## 编程能力对比

| 维度 | Claude (3.5+) | GPT-4o |
|------|--------------|--------|
| 长上下文 | ✅ 200K tokens | ⚠️ 128K tokens |
| 代码生成质量 | 🔥 极强（特别是架构设计） | 🟢 强（日常开发） |
| 多文件编辑 | ✅ 原生支持多文件操作 | 🟡 需要插件 |
| Agent 能力 | ✅ Tool Use 原生支持 | 🟢 Function Calling |
| 安全性 | 🔥 Constitutional AI | 🟢 RLHF |
| 中文能力 | 🟢 良好 | 🔥 优秀 |
| 价格 | 💰 更贵 | 💰 更便宜 |

## 适用场景

**选 Claude 当：**
- 大型代码库重构 / 需要长上下文
- 安全敏感项目（金融、医疗）
- 需要 Agent 自主调用工具链

**选 GPT 当：**
- 快速原型开发 / 小功能实现
- 需要丰富的第三方集成
- 成本敏感的批量任务
