## 我试过的（都失败了）

**Notion 建数据库。**

建了三个 workspace——对三个听起来挺有决心吧？每个填不到 10 条记录就放弃。每次手动分类打标签建关联太累了。真的太累。累到崩溃。崩溃边缘。人的惰性可怕：收集成本大于"以后有用"的预期收益就永远不会发生。Behavioral economics 写明白了。我是反面教材。称职的那种反面教材。反面得不能再反面。丢人现眼。

**Obsidian + 插件。**

配置两天装一堆插件漂亮界面爽三天。然后中文支持一塌糊涂双向链接中文内容经常断掉。本质还是编辑器——不帮你整理只给你更好看的本子。白折腾。两天配置全打水漂。浪费时间。时间就是金钱啊亏大了。心痛。

**直接丢给 ChatGPT/Claude。**

上传文档问问题能用。能用就行了吗？

问题——下次问同样的问题它**重新来过**。没记忆没积累没上下文。像每天早上跟同事打招呼他每天都问"我们认识吗？"

——不对类比不准确。实际比这更蠢。至少同事慢慢认识你。ChatGPT 不会。每次全新白痴。完全不带记忆的蠢。蠢得让人无语。无语凝噎。

这太扯了。受够了这种"伪效率"。工具一大堆效率没涨。焦虑倒是涨了不少。越用越焦虑越焦虑越找工具恶性循环。内耗严重。严重内耗。耗到自己都快没了。

## KnowFlow 的诞生

后来看到 Andrej Karpathy 的 LLM Wiki 想法。Karpathy——前 Tesla AI 总监 OpenAI 联合创始人——提出跟 RAG 完全不同的思路：**别每次查询去原始文档翻先把知识编译好放在那用的时候直接拿。**

RAG 是临时工。LLM Wiki 是全职图书管理员。这个类比打动了我。决定自己做。反正闲着其实并不闲项目堆着。瞎折腾呗。折腾出来算赚折腾不出来也不亏。就当练手。练手也是学。不亏。

### 技术选型（每一步都是取舍）

| 选型 | 选择 | 为什么 |
|------|------|--------|
| 运行时 | Node.js + Python 混合 | 熟悉 JS Python 数据处理强 |
| AI 引擎 | 智谱 AI | 中文 embedding 比 OpenAI 好 免费够用 |
| 存储 | Markdown + SQLite | 本地优先 不需要数据库服务 |
| 可视化 | D3.js 力导向图 | 知识图谱需要直观展示 |
| CLI 框架 | Commander.js | 轻量够用 |

说实话智谱 AI 是最意外收获。一开始想用 OpenAI 跑一圈——不对，准确地说是根本没想别的，直接就上 OpenAI 了。跑完数据后发现中文效果一般般。换成智谱再跑一遍——**好多了**。实测结果。不服自己跑一遍试试。数据不会骗人大概应该。

## 它现在能做什么

四个命令：

`knowflow ingest URL` — 丢 URL 进去自动下载→AI 提取实体概念→生成 Wiki 页面→更新知识图谱→建向量索引。一条龙。甩手掌柜。虽然掌柜偶尔偷懒。

`knowflow graph` — 浏览器打开 D3.js 知识图谱。节点实体概念边关联。节点大小=重要度颜色=类型可拖拽缩放点击查看详情。看着知识变成图挺有成就感——145 个节点、791 条边，终于像个真正的知识网络了。

`knowflow query "问题"` — 基于编译好的 Wiki 内容回答标注来源。不去原始文档翻从整理好的知识库调取。快很多。体验上了一个台阶。终于不用翻箱倒柜了。翻了也找不到那种。

`knowflow health` — 健康检查：文章数节点数向量索引状态 API 连接。体检。定期体检好习惯虽然我不体检。不体检心也虚。

## 踩过的坑

**Shell 脚本 SIGABRT 崩溃。** pipeline.sh 跑到第五步偶尔崩溃 exit code 134。子进程超时信号传递问题。加了 `timeout 30s` 容错把崩溃当 WARN 不是 FAIL。简单粗暴有效。优雅个屁能跑就行。工程嘛能跑就是好工程（狗头）。能跑就是胜利。胜利来之不易。

**中文 Wiki 链接断裂。** 批量生成页面大量 `[[entities/,]]` `[[concepts/,]]` 空链接——AI 提取时生成空实体名。清理脚本修掉 12 个文件。bug 不致命但很烦——不报错 wiki 到处死链接。静默失败最难排查——没有之一。找这 bug 的时间够重写 extract 了。真的够。浪费生命。

**npm 名字被抢。** 一开始想叫 mindwiki npm 被占了。换了 knowflow GitHub npm 都可用。**教训：名字第一时间占位。** 别等做完再想名字太晚了。互联网世界慢一秒就没了一真不夸张。慢了被人抢了只能认栽。认栽感觉糟糕。糟糕透了。
