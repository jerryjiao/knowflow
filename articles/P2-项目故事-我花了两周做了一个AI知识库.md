# 我花了两周做了一个 AI 知识库，这是效果

## 先说结果

做了两周。Wiki 里躺着 **148 篇**文章，知识图谱 **145 个节点、791 条关联关系**——说"两周"其实不准确，断断续续加起来大概十天？剩下的时间在踩坑。好累。

不多。但它是活的。活的就好死代码没意义。没意义就是垃圾。垃圾该扔。死了就白写了。白写最伤心。伤心欲绝 凉了。

## 起因：我的信息焦虑

我是一个前端开发，最近在考虑转型。转型的第一步不是学新框架——是**把脑子里和收藏夹里的碎片信息变成可用知识**。

我关注的东西很杂：AI 工具、开发者效率、独立开发、技术创业、前端框架……每天刷 Twitter 看公众号读技术博客看到好的就收藏。

然后？然后就没有然后了。

微信收藏 ✏️**[请填写真实数量]** 篇文章上次打开是三月份。浏览器书签？✏️**[请填写浏览器书签大概数量]** 个，90% 连标题都想不起来为什么存。这太荒谬了。

OpenText 报告说 80% 知识工作者经历信息过载。我至少占 80.1%。重症患者。得治。

问题不在没有好信息。在**信息进来了但从来没被消化过**。就像一直吃东西从不消化——到头来撑得难受营养不良还胖。

## 我试过的（都失败了）

**Notion 建数据库。** 建了三个 workspace——听起来挺有决心吧？每个填不到 10 条记录就放弃。每次手动分类打标签建关联太累了。人的惰性可怕：收集成本大于"以后有用"的预期收益就永远不会发生。

**Obsidian + 插件。** 配置两天装一堆插件漂亮界面爽三天。然后中文支持一塌糊涂双向链接中文内容经常断掉。本质还是编辑器——不帮你整理只给你更好看的本子。

**直接丢给 ChatGPT/Claude。** 上传文档问问题能用。但问题——下次问同样的问题它**重新来过**。没记忆没积累没上下文。像每天早上跟同事打招呼他每天都问"我们认识吗？"

这太扯了。受够了这种"伪效率"。工具一大堆效率没涨。焦虑倒是涨了不少。

## KnowFlow 的诞生

后来看到 Andrej Karpathy 的 LLM Wiki 想法。Karpathy——前 Tesla AI 总监 OpenAI 联合创始人——提出跟 RAG 完全不同的思路：**别每次查询去原始文档翻先把知识编译好放在那用的时候直接拿。**

RAG 是临时工。LLM Wiki 是全职图书管理员。这个类比打动了我。决定自己做。

### 技术选型（每一步都是取舍）

| 选型 | 选择 | 为什么 |
|------|------|--------|
| 运行时 | Node.js + Python 混合 | 熟悉 JS Python 数据处理强 |
| AI 引擎 | 智谱 AI (GLM-4-Flash) | 中文 embedding 效果好 免费够用 |
| 存储 | Markdown + JSON | 本地优先 不需要数据库服务 |
| 可视化 | vis-network (vis.js) | 知识图谱力导向图可视化 |
| 向量检索 | 智谱 embedding-3 | 中文语义搜索 |
| CLI 框架 | Commander.js | 轻量够用 |

智谱 AI 是最意外收获。一开始想直接上 OpenAI，跑完数据后发现中文效果一般般。换成智谱再跑一遍——**好多了**。实测结果。

## 它现在能做什么

四个命令：

`knowflow ingest URL` — 丢 URL 进去自动下载→AI 提取实体概念→生成 Wiki 页面→更新知识图谱→建向量索引。一条龙。

`knowflow graph` — 浏览器打开 **vis.js 力导向图**知识图谱。节点=实体/概念，边=关联关系。节点大小=引用次数，颜色=类型，可拖拽缩放点击查看详情。

`knowflow query "问题"` — 基于编译好的 Wiki 内容回答标注来源。不去原始文档翻从整理好的知识库调取。

`knowflow health` — 健康检查：文章数、节点数、向量索引状态、API 连接。

## 踩过的坑

**Shell 脚本 SIGABRT 崩溃。** pipeline.sh 跑到第五步偶尔崩溃 exit code 134。子进程超时信号传递问题。加了 `timeout 30s` 容错把崩溃当 WARN 不是 FAIL。

**中文 Wiki 链接断裂。** 批量生成页面大量 `[[entities/,]]` `[[concepts/,]]` 空链接——AI 提取时生成空实体名。清理脚本修掉 12 个文件。静默失败最难排查。

**npm 名字被抢。** 一开始想叫 mindwiki npm 被占了。换了 knowflow GitHub npm 都可用。**教训：名字第一时间占位。**
